There are five most imp fetching methods of DOM Elements.
1.  querySelector():- It is a method in JavaScript used to select the first element within the document that matches a specified CSS selector or group of selectors.
                      It returns the first matching element or null if no elements match the selector.
2.  querySelectorAll():- It is a method in JavaScript that returns a static NodeList (which is similar to an array)
                         of all elements within the document that match a specified CSS selector or group of selectors.
